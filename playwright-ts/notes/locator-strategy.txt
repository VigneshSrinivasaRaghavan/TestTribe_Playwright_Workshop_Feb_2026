input[name="user-name"]

// Strategy for class varies compared to id,name
1. For Css using the dot operator --> .classValue

2. CSS - Use the filtering way --> starts-with, ends-with, contains
input[class="oxd-input oxd-a oxd-b oxd-c oxd-input--active"]
stats-with ---> ^ --> input[class^="oxd-input"]
ends-with ---> $ --> input[class$="oxd-input--active"] or input[class$="active"] 
contains-with --> * --> input[class*="oxd-input"]

3. Combine css/xpath with other attributes
input[class^="oxd-input"][placeholder="Username"]
//input[@class="oxd-input oxd-input--active"][@placeholder="Username"]

Order of the locator present in DOM
(//input[@class="oxd-input oxd-input--active"][@placeholder="Username"])[1] --> Highlighting the 1st element
(//input[@class="oxd-input oxd-input--active"][@placeholder="Username"])[2] --> Highlighting the 2nd element
